#### 题目练习

### 反转字符串

- [题目链接](https://leetcode.cn/problems/reverse-string/submissions/)

- 解题思路：使用双指针left right  互换位置，当left<=right时候循环条件结束

- 代码示例：

```Go
// 反转字符串
func reverseString(s []byte) {
   left := 0
   right := len(s) - 1
   for left <= right {
      s[left], s[right] = s[right], s[left]

      left++
      right--
   }

   fmt.Println(string(s))
}
```
